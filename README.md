# AxeXEC Release

This repo contains the **app recipe files** for AxeXEC (built for 5tratumOS and compatible self-hosted app platforms).

It must **not** contain any application source code.

The build repo publishes the UI image to GHCR:
- `ghcr.io/willitmod/axexec-app:<tag>`

## Current status

- Current dev-store package version: `0.1.10-dev`
- Mirrored dev-store package: `WillItMod/umbrel-dev-community-store/willitmod-dev-xec`
- Miner endpoint: `stratum+tcp://<host-ip>:4321`
- For the cross-project version matrix and release/changelog pointers, see `https://github.com/WillItMod/AxeSuite/blob/main/docs/releases.md`.
